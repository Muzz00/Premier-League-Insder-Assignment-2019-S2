Visual Design Page



My website has been layed out such that it is User Friendly and easily accessible for users to use and read. I have chosen colours for the website and images such that it wouldn't not put off the user whilst maintaing a smooth transition to within and beteween the websites. I went with a darker theme for the navigation bar and the footer so that users are able to grasp the content on the websites as soon as they enter it.


The font family I have used for the text are the default fonts of the browser. This is because a browser maybe optimised to handle specifc fonts. For Google chrome on windows the font is Arial, Times New Roman, Courier New whils on a Mac OS X it would be Helvetica, Times-Roman, Courier.
