Mutimedia Elements Page



There are many multimedia elements used in my website such as image tags, videos and other HTML tags. Images placed on the website are relevant to the specific page and have been if/where necessary.


The JavaScript used on my websites are on the Premier League and the Champions League Spot page. The JavaScript is located on in the script tags within the head tags of the respective websites. They get the document by element id (in this case "slide") and create a slideshow using list. The JavaScript function is called in the body tag when the page is loaded. The images are set to rotate every 3000 milliseconds (3 seconds).
