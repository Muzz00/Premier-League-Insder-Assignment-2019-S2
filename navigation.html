Navigation



Navigation through this website is easy and user friendly. Above is the navigation bar for all 14 pages. Once any icon on the Navigation is clicked it will take you to the desired page. To view which current page you are on, the page number will be highlighted in a darker colour to make it more identifiable.


On the bottom right of the page there are two icons. When the left arrow is clicked it will take you to the previous page and the right arrow will take you the next page. The arrows are chornologically ordered similar to that of the Navigation bar.
